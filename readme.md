# ALM Task Manager

ALM Task Manager is a project managemnet system built on Laravel 4.2.*

# Features
  - User / account creation
  - Client creation
  - Project creation
  - Task creation 
  - Assigning weights to tasks
  - file uploads
  - project sharing




#Installation
-   Clone the repo
-   Run composer install
-   Run php artisan migrate
-   Run php artisan db:seed
-   Start developing!
-   easy to setup
